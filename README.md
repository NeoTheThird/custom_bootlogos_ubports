Custom Bootlogos ubports
==========================
Changed Boot_logos For Ubuntu Touch Ubports

Nexus 5
=======
fastboot flash imgdata imgdata.img

Fairphone 2
===========
fastboot flash splash splash.img

Meizu MX4
=========

Call MT6595_Android_scatter.txt and flash only logo.
